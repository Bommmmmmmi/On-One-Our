권악징선 수배자 명단

바이스 Vice

분류 : 勸 (권) 등급 : ● ● ● ○ ○ 이름 : 바이스 (VICE) 능력 : Temptation 수갑을 채우면 자신에게 복종하도록 만들 수 있음. 단거리 능력이라 무술에 뛰어남. 주로 쇠사슬을 던져 몸을 결박한 후 수갑을 채우는 수법을 사용함. 부작용 : 복종자들과 쇠사슬로 이어져 있어 활동에 제약이 있음. 체포 지침 : 최대한 원거리에서 대치, 사슬을 피할 것.

아크 Ark

분류 : 惡 (악) 등급 : ● ● ● ● ● 이름 : 아크 (ARK) 능력 : Run wild 망치를 사용해 마구잡이로 부수고 다님. 안대를 벗으면 신체능력이 비정상적으로 상승함. 부작용 : 힘을 사용한 만큼 의식을 잃음. 체포 지침 : 협상 불가능. 행동 불능 및 제압.

퓨너 Punar

분류 : 懲 (징) 등급 : ● ● ● ● ● 이름 : 퓨너 (Punar) 능력 : 懲罰 최대 20m 내외, 시야에 보이는 지정한 물질과 공간을 강제적으로 수축시켜 밀도를 비정상적으로 끌어올림. 대상의 형태, 크기, 구조를 무시하고 억지로 압축시킴. 부작용 : 힘을 사용할수록 몸과 감각에 부담감 누적. 신체적•정신적 고통 수반. 체포 지침 : 원거리 제압, 에너지 차단 장치 필요.

젠 Zen

분류 : 善 (선) 등급 : ● ● ● ● ○ 이름 : 젠 (ZEN) 능력 : 吸光 주변의 빛을 흡수함. 범위는 달이 가려진 정도에 따라 달라짐. 보름달의 경우 하나의 지역 범위까지 확장됨. 그 외에 어둠 속을 자유롭게 볼 수 있음. 부작용 : 능력을 사용한 시간 만큼 눈이 멀게 됨. 체포 지침 : 특수 제작된 안경을 사용하여 신속히 포획.
